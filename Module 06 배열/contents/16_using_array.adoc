= 배열 사용

이 단원에서는 배열을 사용하는 방법과 배열을 메소드에 매개변수로 전달하는 방법을 학습합니다.

배열 인스턴스 요소의 기본값을 제어하고 인스턴스의 복사본을 만들어 다른 배열에 할당하는 방법, 배열 사용과 조작을 위한 많은 기능을 제공하는 java.util.Arrrays 클래스를 사용하는 방법 및 foreach 문을 사용하여 배열의 요소를 읽는 방법을 배웁니다.
