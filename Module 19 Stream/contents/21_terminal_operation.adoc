= 최종 연산

**Java Stream의 최종 연산(Terminal Operations)**은 스트림의 데이터를 처리한 결과를 반환하거나 실행을 완료하는 작업입니다. 

이 단원에서는 Stream API의 최종 연산에 대해 알아봅니다.
