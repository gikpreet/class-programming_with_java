= Generics 기본

Java Generics는 컴파일 시점에 타입을 지정할 수 있는 기능으로, 타입 안정성과 코드 재사용성을 제공하기 위해 도입되었습니다. Generics는 주로 클래스, 메서드, 인터페이스에서 사용되며, 데이터를 저장하거나 처리할 때 타입 안정성을 확보합니다.

이 단원에서는 Generics의 기본적인 내용에 대해 학습합니다.
