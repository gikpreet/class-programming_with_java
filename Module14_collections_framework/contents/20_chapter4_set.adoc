= Chapter 4: Set
Set은 수학에서 말하는 집합과 유사한 자료구조를 표현합니다. Set 인터페이스는 집합과 유사하게 데이터의 중복을 허용하지 않으며 저장 순서가 유지되지 않습니다. 이 장에서는 Set 인터페이스의 구현체인 SortedSet, HashSet, TreeSet에 대해 학습합니다.
