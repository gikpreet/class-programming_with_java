= Chapter 1: 값 타입과 참조 타입

모든 변수에는 변수에 저장할 값의 범위를 결정하는 타입이 지정됩니다. Java는 컴파일 시 변수에 저장된 값이 항상 적절한 형식임을 보장하는 type-safe한 언어입니다.

이 단원에서는 값 타입과 참조 타입에 대해 알아보고 값 타입 변수에 값을 저장할 수 있는 데이터 유형을 배웁니다.
