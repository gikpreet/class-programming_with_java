= Java 패키지

Java의 패키지는 비슷한 성격의 클래스들을 모아 놓은 디렉토리입니다. Java는 클래스들을 여러 이름공간으로 정리하며, 소속된 각 타입에 대해 고유한 이름 공간들을 제공합니다. 이 장에서는 Java 패키지의 동작 방법과 Java에서 제공되는 클래스들을 제공하는 패키지를 알아봅니다.
