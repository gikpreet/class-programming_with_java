= 추상클래스

추상 클래스는 파생된 클래스에서 완료할 수 있는 부분적인 구현을 제공하는데 사용됩니다. 추상 클래스는 여러 서브 클래스에서 재사용할 수 있는 코드를 상속하여 재사용을 제공한다는 데에서 유용합니다. 이 장에서는 추상 클래스를 생성하고 사용하는 것을 배웁니다.
